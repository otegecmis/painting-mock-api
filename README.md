## Painting Mock API

- A `mock API` built with `C#` and `.NET Core`, using `EF Core`, that offers CRUD operations to manage paintings, artists and museums.

### Getting Started

1. Clone the repository.
```sh
git clone https://github.com/otegecmis/painting-mock-api.git
```

2. Navigate to the MockAPI directory.
```sh
cd painting-mock-api/MockAPI
```

3. Restore dependencies.
```sh
dotnet restore
```

4. Run the application.
```sh
dotnet run
```

### API Documentation

Access `Swagger UI` at `/swagger` to test the API.
